`Icons` is completely use from Lucide icon

```bash
yarn add @next-dev/icons
yarn add  react-native-svg (React Native only)
```

```tsx | pure
import { MemoryStick } from '@next-dev/icons'

export default () => {
  return (
    <>
      <MemoryStick />
    </>
  )
}
```
