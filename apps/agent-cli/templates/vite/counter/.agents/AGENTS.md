# {{PROJECT_NAME}}

A Vite + React + TypeScript project with AI agent configuration.

## Project Structure

- `src/` — Source code (React components, hooks, utils)
- `public/` — Static assets
- `.agents/` — Universal AI agent configuration

## Conventions

- Use **React** with functional components and hooks
- Use **TypeScript** everywhere
- Use **Vite** for development and builds

## Commands

| Command          | Description               |
| ---------------- | ------------------------- |
| `npm run dev`    | Start dev server          |
| `npm run build`  | Build for production      |
| `agent-cli sync` | Sync agent config to IDEs |
