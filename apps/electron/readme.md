
# Electron app for Next Docs

This Electron app launches the Next.js docs (`@rnr/docs`) and embeds it in a desktop window.

Development:

1. Install dependencies: `pnpm install`
2. Start Electron (it will start Next dev if not running): `pnpm --filter @rnr/electron dev`

The app will open `http://localhost:3000` inside Electron.
