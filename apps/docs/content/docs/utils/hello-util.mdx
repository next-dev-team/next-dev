## Overview

Utils are utility functions that provide common functionality without UI components. These functions demonstrate input/output examples rather than visual previews.

## Available Utils

### Hello Utility

A simple utility for generating greeting messages.

<HelloDemo />

## Usage

All utils can be imported from the `@rnr/utils` package:

```typescript
import { hello, helloAsync } from '@rnr/utils';

// Synchronous usage
const greeting = hello("World");
console.log(greeting); // "Hello, World! ðŸ‘‹"

// Asynchronous usage
const asyncGreeting = await helloAsync("React Native");
console.log(asyncGreeting); // "Hello, React Native! ðŸ‘‹"
```
