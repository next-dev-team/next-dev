## Getting Started with the template

Go to this repository's [github page](https://github.com/next-dev-team/next-dev) and click on the `Use this template` button. This will create a new repository with the template.

### Install the dependencies

<BashCommand command={`pnpm install`} />

- Run the development server for both apps:

<BashCommand command={`pnpm dev`} />

- Run the development server for the docs app:

<BashCommand command={`pnpm dev:docs`} />

- Run the development server for the showcase app:

<BashCommand command={`pnpm dev:showcase`} />

- Build the registry:

<BashCommand command={`pnpm registry`} />

- Format the code:

<BashCommand command={`pnpm format`} />
